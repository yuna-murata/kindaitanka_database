# kindaitanka_database
[近代短歌データベース](http://kindaitankadatabase.com/)に関するデータを掲載しています。

データの利用に際しては、上記サイトの「データの利用について」をご参照ください。  

## リポジトリのデータについて
### itiji.txt

- 検索システムSolrに登録している異体字対応表です。[デジタル版渋沢栄一伝記資料　新旧文字置換対照表](https://eiichi.shibusawa.or.jp/denkishiryo/digital/main/index.php?kanji)を改変したものです。

### 底本.csv 

- テキストデータの底本情報について記載しています。

### 歌集制作年・出版年.csv

- 歌集ごとの制作年、出版年をまとめています。
- 凡例
  - ●yyyy-yyyy：範囲で制作年/出版年が判明したもの。年数による検索の対象だが、グラフ集計は対象外
  - ●0000：制作年/出版年が不明のもの。年数検索、グラフ集計ともに対象外
  - yyyy-yyyy：制作年/出版年が判明し、それぞれの年を付与しているもの。年数検索、グラフ集計の対象
  - [yyyy]：制作年/出版年を補記したもの。年数検索、グラフ集計の対象
  - 制作年を補記：草稿等（未刊行）の歌で、出版年として制作年を補記しているもの。年数検索、グラフ集計の対象
